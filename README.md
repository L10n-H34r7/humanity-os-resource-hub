# 🌍 Humanity OS

**Open source seeds for a world without money.**

> *"Another world is not only possible, she is on her way. On a quiet day, I can hear her breathing."* — Arundhati Roy

Humanity OS (Humanity Operating System / Humanity Open Source) is the idea that we can live together in an abundant world without money, in anarcho-collective ways. We have all the knowledge to make a better world — what's missing are the **links**.

This is a central hub connecting open source projects, real-world collectives, legal frameworks, and starter packs proving that another world is not utopia — **it already exists**.

## 🏗️ Deploy Your Own

This site is a single HTML file. To deploy:

### GitHub Pages (simplest)
1. Fork this repo
2. Go to **Settings → Pages → Source** → select `main` branch, `/ (root)` folder
3. The `index.html` in the root is the complete site
4. Your site will be at `https://yourusername.github.io/repo-name/`

### Anywhere else
The `index.html` file is completely self-contained (all JS/CSS inlined). Upload it literally anywhere — any web server, IPFS, Tor, a USB stick. It just works.

## 📜 The Six Pillars

1. **🔄 Gift Economy** — The oldest form of exchange. Trust, reciprocity, abundance.
2. **🏛️ Collective Governance** — Direct democracy works. People can govern themselves.
3. **🌱 Commons & Shared Ownership** — Land and production owned collectively, off the market.
4. **💡 Open Source Everything** — Knowledge, tools, designs shared freely.
5. **⚖️ Mutual Aid** — From each according to ability, to each according to need.
6. **⚡ Post-Scarcity Tech** — We produce enough for everyone. Scarcity is manufactured.
